# Phones Project

Простой каталог смартфонов на Django.

Проект позволяет просматривать список телефонов, фильтровать их по цене и названию, а также смотреть детальную информацию о каждом смартфоне.

## Возможности

- Просмотр каталога телефонов
- Сортировка по:
  - Названию (A–Z)
  - Цене (сначала дешевые)
  - Цене (сначала дорогие)
- Детальная страница каждого телефона

## Доступные URL
- /stars/catalog/ — Каталог телефонов
- /stars/catalog/\<slug>/ — Детальная страница телефона


## Запуск проекта 

### Для запуска проекта необходимо

Склонировать репозиторий

```bash
git clone git@github.com:REBIZ1/phones_project.git
```

Создать и активировать виртуального окружения
Windows
```bash
.venv\Scripts\activate
```

Linux / macOS
```bash
source .venv/bin/activate
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

Выполнить следующие команды:

- Команда для создания таблиц в бд
```bash
python manage.py migrate
```

- Команда для загрузки данных из .csv:
```bash
python manage.py load_phones
```

- Команда для запуска приложения
```bash
python manage.py runserver
```
